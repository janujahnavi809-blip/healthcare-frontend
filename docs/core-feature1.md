# Core Feature 1

## Diabetes Prediction Model

This feature uses Logistic Regression to predict whether a patient has diabetes based on healthcare measurements.

### Inputs

* Pregnancies
* Glucose
* BloodPressure
* SkinThickness
* Insulin
* BMI
* DiabetesPedigreeFunction
* Age

### Output

* 0 = No Diabetes
* 1 = Diabetes

### Edge Cases

* Empty input handled
* Invalid data type handled

### Testing

Feature tested with five different scenarios.
