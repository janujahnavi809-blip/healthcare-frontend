# Advanced Feature 1

## Interactive Patient Input System

This feature allows users to enter patient health details dynamically through the keyboard.

## Inputs

- Patient ID
- Pregnancies
- Glucose
- Blood Pressure
- Skin Thickness
- Insulin
- BMI
- Diabetes Pedigree Function
- Age

## Output

The system predicts whether the patient is likely diabetic or non-diabetic.

## Testing

The feature was tested with:
1. Valid numeric inputs
2. Invalid text inputs
3. Empty inputs

## Result

Feature implemented and tested successfully.
